Features

Eight essentials.
Two of them un-copyable.

Overmind closes the gaps with the leading orchestrators on the essentials — without diluting its two differentiators: first-class organizational memory, and a tamper-evident, hash-chained audit trail. Everything below is enforced by the server, not just suggested in a prompt.

Differentiator

Organizational memory

One persistent brain per org, over MCP. Agents recall before working and store what they learn — decisions, fixes, and rationale outlive any single run.

Differentiator

Hash-chained audit

Every action is an append-only, SHA-256-linked event committed with the change it records. GET /audit/verify re-walks the chain and names the exact broken block.

Execution

Isolated git worktrees

Each run gets its own worktree off your repo — parallel agents never collide on a working tree, and every change lands on its own branch, reviewable before it merges.

Governance

Atomic budgets

Spend limits are enforced at checkout, atomically: an agent can't claim a task and overspend in the same breath. The budget check and the claim succeed or fail together.

Governance

Approval gates

Set an agent to propose-only or act-with-approval and it blocks at the gate until you sign off. Autonomy is a structured, server-enforced setting — never just prompt text.

Organization

Org chart

Agents are employees with roles, managers, and reporting lines rooted at you. Hire under any node; the chart is the real structure work and budgets flow through.

Reliability

Heartbeats & recovery

A scheduler wakes agents on a heartbeat, runs them in parallel with timeouts, and recovers stuck or crashed runs — so long-lived work survives restarts and failures.

Onboarding

Guided hiring

A three-level hire dialog — pick, tune, expert — turns an archetype into a configured agent: role, autonomy, review strictness, model, budget. Structured choices map to enforced config.

Enforced, not suggested.  Every option above is a structured setting the Rust server enforces — budgets at checkout, autonomy at the gate, transitions against a mirrored state machine. Prompts describe intent; the server holds the line.
